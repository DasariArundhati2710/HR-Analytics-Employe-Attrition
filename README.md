# HR Analytics – Employee Attrition Analysis

## Project Overview
This project analyzes employee data to identify patterns related to employee attrition.

## Objectives
- Perform data cleaning
- Conduct Exploratory Data Analysis (EDA)
- Visualize workforce trends
- Identify factors affecting employee attrition

## Tools Used
Python  
Pandas  
Matplotlib  
Seaborn  
Jupyter Notebook  

## Dataset
IBM HR Employee Attrition Dataset

## Files in this Repository
HR_Analytics_Employee_Attrition_Analysis.ipynb – Analysis notebook  
WA_Fn-UseC_-HR-Employee-Attrition.csv – Dataset  
HR_Analytics_Employee_Attrition_Report.pdf – Project report

## Conclusion
The analysis highlights key factors such as overtime, job satisfaction, and income that may influence employee attrition.
